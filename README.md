# Gallery
J2EE项目开发与设计大作业

##功能介绍
1.游客/普通用户：搜索艺术作品，查看详情介绍页。查看其他用户分享内容。
2.普通用户：在分享页发表评论/自己拍摄的照片。查看自己分享全部内容。
3.管理员用户：管理用户，管理用户分享内容，编辑/添加数据。

##主要技术
1.JSP
2.JDBC
3.Servlet
4.基础数据库知识
